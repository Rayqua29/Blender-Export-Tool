# Blender Export & Layout Tool

Python tool for Blender asset export and scene layout workflows.

## Features

### One-Click Batch FBX Export
- Export all selected objects as individual FBX files
- Automatic naming from Blender object names

![Export Demo](https://github.com/Rayqua29/Blender-Export-Tool/blob/main/gifs/1.gif)

### Bulk Rescale
- Rescale multiple selected objects simultaneously
- Uniform scaling with single parameter

![Rescale Demo](https://github.com/Rayqua29/Blender-Export-Tool/blob/main/gifs/3.gif)

### Position Reset
- Reset position of selected objects to origin
- One-click cleanup for messy scenes
- Preserves rotation and scale

![Reset Demo](https://github.com/Rayqua29/Blender-Export-Tool/blob/main/gifs/4.gif)

### Grid Layout Tool
- Automatically arrange selected objects in grid pattern
- Customizable spacing (X/Y axis)
- Configurable number of rows

![Grid Layout Demo](https://github.com/Rayqua29/Blender-Export-Tool/blob/main/gifs/2.gif)
